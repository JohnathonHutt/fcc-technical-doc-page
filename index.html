About Ruby

Ruby is a language of careful balance. Its creator, Yukihiro “Matz” Matsumoto, blended parts of his favorite languages (Perl, Smalltalk, Eiffel, Ada, and Lisp) to form a new language that balanced functional programming with imperative programming.

He has often said that he is “trying to make Ruby natural, not simple,” in a way that mirrors life.

Building on this, he adds:


"Ruby is simple in appearance, but is very complex inside, just like our human body."

Hello World

Text or strings are displayed in Ruby by using puts (for put string), which prints the string and adds a newline to the end of the string.

puts "Hello World"

Alternatively, the print command can be used. The difference between these two commands is that print does not add a newline to the end of the statement.

print "hello World"

Syntax

Ruby is known for it's elegant syntax - take one line if statements for example.

puts "Get some coffee" if alertness == 0

Here is an example of a Ruby method. Notice the absence of curly braces and semicolons.

def say_hello(name)
return "Hello, " + name
end

One of the many cool Ruby features are implicit return statements. In Ruby methods return the last statement that is evaluated, meaning that the previous method can be simpified to...

def say_hello(name)
"Hello, " + name
end
